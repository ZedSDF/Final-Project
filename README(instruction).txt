Инструкция по использованию:

Добавление Сотрудников:
1.Введите данные сотрудника в поля Фамиля, Имя, Телефон, Email, Зарплата
2.Нажмите кнопку "Добавить сотрудника" и сотрудник отобразится в верхнем кошке

Обновление сотрудников:
1.Выберете сотрудника в списке в верхнем окошке
2.Введите его ID в поле "ID сотрудника для обновления/удаления"
3.Изменити данные которые вам нужно изменить
4.Нажмите "Обновить сотрудника" и вы увидите измененные данные в верхнем окошке

Удаление сотрудников:
1.Введите ID сотрудника в поле "ID сотрудника для обновления/удаления"
2.Нажмите "Удалить сотрудника" и вы увидете что его больше нет в верхнем окошке
